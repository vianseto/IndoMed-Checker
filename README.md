# indo-med-checker

Proyek ini mengembangkan alur pembacaan teks dari foto kemasan obat (OCR), pencocokan dengan data referensi, serta peninjauan petunjuk interaksi berdasarkan teks yang tersedia. Repositori ini masih dalam tahap awal.

**Lingkungan pengembangan:** Python 3.10 atau lebih baru (diuji pada Windows). Penggunaan pada sistem operasi lain dapat memerlukan penyesuaian dependensi.

**Peringatan:** alat ini tidak menggantikan konsultasi dokter atau apoteker; dimaksudkan untuk edukasi dan penelitian.

Pertanyaan atau masukan dapat disampaikan melalui *Issues* pada repositori ini.
